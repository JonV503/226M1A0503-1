Functions

Function - is a reusable code-block that will be executed by an event, or when the function is called. Note : To keep the browser from executing a script when the page loads, we can put our script into a function. we may call a function from anywhere within the page (or even from other pages if the function is embedded in an external .js file). Functions can be defined both in the and in the section of a document. Note : However, to assure that the function is read/loaded by the browser before it is called, it could be wise to put it in the section.
